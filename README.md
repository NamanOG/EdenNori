## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/NamanOG/EdenNori.git
cd EdenNori
```

2. **Install dependencies**
```bash
pnpm install
# or
npm install
```

3. **Run development server**
```bash
pnpm dev
# or
npm run dev
```

4. **Open in browser**
```
http://localhost:3000
```

### Build for Production

```bash
pnpm build
pnpm start
```
